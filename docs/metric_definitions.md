# Metric Definitions

## latest_price
Most recent observed listed price for a tracked product from source.

## average_category_price
Average observed price for all tracked products in a category on a given date.

## price_change_absolute
Current price minus earliest observed price in analysis window.

## price_change_percent
(current price - earliest price) / earliest price * 100

## observation_count
Number of price observations stored for a product.
